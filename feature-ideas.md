# ToDoアプリ 機能拡張アイデア

## 1. 基本機能の強化

| # | 機能 | 説明 | 優先度 |
|---|------|------|--------|
| 1-1 | 優先度 | 高/中/低の設定と並び替え | - |
| 1-2 | タグ/ラベル | 複数タグで柔軟に分類 | - |
| 1-3 | サブタスク | ToDoを階層化して細分化 | - |

## 2. ユーザー体験向上

| # | 機能 | 説明 | 優先度 |
|---|------|------|--------|
| 2-1 | 検索機能 | キーワードでタスクを検索 | - |
| 2-2 | ソート機能 | 期限日・優先度・作成日で並び替え | - |
| 2-3 | ドラッグ&ドロップ | タスクの順序を自由に変更 | - |
| 2-4 | ダークモード | UIテーマ切替 | - |

## 3. コラボレーション

| # | 機能 | 説明 | 優先度 |
|---|------|------|--------|
| 3-1 | コメント | タスクへのメモ・コメント追加 | - |

## 4. 分析・可視化

| # | 機能 | 説明 | 優先度 |
|---|------|------|--------|
| 4-1 | ダッシュボード | グラフで進捗を可視化 | - |
| 4-2 | カレンダービュー | 期限をカレンダーで表示 | - |
| 4-3 | ガントチャート | プロジェクトのタイムライン | - |

## 5. チケット詳細画面

### 5.1 詳細モーダル/パネル

| # | 機能 | 説明 | 優先度 |
|---|------|------|--------|
| 5-1-1 | モーダル詳細表示 | クリックでオーバーレイ表示、背景クリックで閉じる | - |
| 5-1-2 | サイドパネル | 右側にスライドイン、一覧を見ながら編集可能 | - |
| 5-1-3 | インライン展開 | カード内で詳細が展開/折りたたみ | - |

### 5.2 詳細画面の編集機能

| # | 機能 | 説明 | 優先度 |
|---|------|------|--------|
| 5-2-1 | インライン編集 | フィールドをクリックで直接編集 | - |
| 5-2-2 | フォーム編集 | 編集モードに切り替えて全項目変更 | - |
| 5-2-3 | オートセーブ | 変更時に自動保存（手動保存不要） | - |

### 5.3 詳細画面で見たい情報

| # | 機能 | 説明 | 優先度 |
|---|------|------|--------|
| 5-3-1 | ステータスバッジ | 完了/未完了/期限切れを色で表示 | - |
| 5-3-2 | 残り日数表示 | 期限まであと○日 | - |
| 5-3-3 | 案件情報 | 所属プロジェクトへのリンク | - |
| 5-3-4 | 関連タスク | 同じ案件の他タスクを表示 | - |

### 5.4 操作性向上

| # | 機能 | 説明 | 優先度 |
|---|------|------|--------|
| 5-4-1 | キーボードショートカット | Escで閉じる、矢印で前後移動 | - |
| 5-4-2 | スワイプ操作 | モバイルで左右スワイプで前後移動 | - |
| 5-4-3 | クイックアクション | ワンクリックで完了/案件変更 | - |

### 5.5 追加項目（将来的に）

| # | 機能 | 説明 | 優先度 |
|---|------|------|--------|
| 5-5-1 | コメント/メモ欄 | タスクに関する補足情報を記録 | - |
| 5-5-2 | 添付ファイル | 関連ファイルをアップロード | - |
| 5-5-3 | チェックリスト | サブタスクをチェックボックスで管理 | - |

## 除外した機能

- 認証・認可
- メール通知
- 繰り返しタスク
- 履歴/監査ログ
