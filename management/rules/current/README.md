## ルール更新時の注意事項

### 更新前

1. 現在のルールファイルを `../archive/YYYYMMDD_*.md` にコピー
2. 変更理由・背景をコミットメッセージに記載

### 更新後

1. 関連チームメンバーへの周知
2. 過去のチケットへの影響確認
