# 本番コード

アプリケーションの本番コードを格納するフォルダ。

## フォルダ構成

```
main/
├── java/                   # Javaソースコード
│   └── com/example/demo/   # パッケージ
└── resources/              # 設定ファイル・静的リソース
    ├── static/             # 静的HTMLファイル
    └── mapper/             # MyBatis Mapper XML
```
