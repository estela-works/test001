# Backend

Spring Boot バックエンドアプリケーション。

## 概要

| 項目 | 内容 |
|------|------|
| フレームワーク | Spring Boot 3.2 |
| 言語 | Java 17 |
| ORM | MyBatis |
| データベース | H2 Database (ファイルモード) |

## ディレクトリ構成

```
backend/
└── main/
    ├── java/com/example/demo/   # Javaソースコード
    └── resources/               # リソースファイル
```
