# マスタデータ

E2Eテストのマスタデータを格納するフォルダ。

## フォルダ構成

```
master/
├── screen-spec/           # 画面仕様JSON
├── code-samples/          # コードサンプル・関数リファレンス
└── test-scenarios/        # テストシナリオ（Markdown）
```

## フォルダ説明

| フォルダ | 説明 |
|---------|------|
| [screen-spec/](screen-spec/) | 画面から自動抽出した仕様JSON |
| [code-samples/](code-samples/) | Playwright関数リファレンス・サンプルコード |
| [test-scenarios/](test-scenarios/) | テストシナリオ（Markdown形式） |
