# テストシナリオ

テストシナリオ（Markdown形式）を格納するフォルダ。

## ファイル構成

| ファイル | 画面 |
|---------|------|
| [todos-scenarios.md](todos-scenarios.md) | ToDo管理画面テストシナリオ |
| [projects-scenarios.md](projects-scenarios.md) | 案件一覧画面テストシナリオ |
| [users-scenarios.md](users-scenarios.md) | ユーザー管理画面テストシナリオ |

## シナリオ形式

各シナリオには以下を記載:
- テストケースID
- テスト目的
- 前提条件
- テスト手順
- 期待結果
