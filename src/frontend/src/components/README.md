# components

再利用可能なVueコンポーネント。

## ディレクトリ構成

| フォルダ | 説明 |
|---------|------|
| `common/` | 共通コンポーネント（LoadingSpinner, ErrorMessage） |
| `todo/` | ToDo関連コンポーネント |
| `project/` | プロジェクト関連コンポーネント |
| `user/` | ユーザー関連コンポーネント |

## コンポーネント一覧

### common/
- `LoadingSpinner.vue` - ローディング表示
- `ErrorMessage.vue` - エラーメッセージ表示

### todo/
- `TodoForm.vue` - ToDo入力フォーム
- `TodoList.vue` - ToDoリスト表示
- `TodoItem.vue` - ToDoアイテム
- `TodoStats.vue` - 統計表示
- `TodoFilter.vue` - フィルター

### project/
- `ProjectForm.vue` - プロジェクト入力フォーム
- `ProjectCard.vue` - プロジェクトカード

### user/
- `UserForm.vue` - ユーザー入力フォーム
- `UserCard.vue` - ユーザーカード
