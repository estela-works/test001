# src

Vue.js フロントエンドのソースコード。

## ディレクトリ構成

| フォルダ | 説明 |
|---------|------|
| `assets/` | 画像・スタイルなどの静的アセット |
| `components/` | 再利用可能なVueコンポーネント |
| `composables/` | Composition API関数（カスタムフック） |
| `router/` | Vue Router設定 |
| `services/` | APIサービス層 |
| `stores/` | Pinia状態管理ストア |
| `types/` | TypeScript型定義 |
| `views/` | ページコンポーネント |

## 主要ファイル

- `App.vue` - ルートコンポーネント
- `main.ts` - アプリケーションエントリーポイント
- `style.css` - グローバルスタイル
