## 3. 構成要素

| 要素ID | 要素名 | 種別 | CSSクラス | 説明 |
|--------|--------|------|-----------|------|
| E-001 | タイトル | テキスト | `.header h1` | アプリケーション名「Spring Boot App」 |
| E-002 | サブタイトル | テキスト | `.header .subtitle` | 「タスク管理アプリケーション」 |
| E-003 | チケット管理カード | リンクカード | `.nav-card.card-todo` | チケット管理画面への遷移カード |
| E-004 | 案件管理カード | リンクカード | `.nav-card.card-project` | 案件管理画面への遷移カード |
| E-005 | ユーザー管理カード | リンクカード | `.nav-card.card-user` | ユーザー管理画面への遷移カード |
| E-006 | フッター | テキスト | `.footer` | コピーライト表示 |

### 3.1 カードUI仕様

各ナビゲーションカードは以下の仕様で実装：

| 項目 | 仕様 |
|------|------|
| ホバー効果 | `transform: translateX(5px)` + `box-shadow` 強調 |
| ホバー時ボーダー | 左側に4pxのカラーボーダー表示 |
| チケット管理カード色 | `#17a2b8`（シアン） |
| 案件管理カード色 | `#28a745`（グリーン） |
| ユーザー管理カード色 | `#fd7e14`（オレンジ） |
| アイコン | 各カードに絵文字アイコン表示 |
