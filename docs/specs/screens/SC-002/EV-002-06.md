| 項目 | 内容 |

| イベント名 | ToDo削除 |
| 画面 | SC-002: ToDoリスト画面 |

---

## 2. 処理フロー

```
1. 削除ボタンクリック
2. 確認ダイアログ表示
   └→「このToDoを削除しますか？」
3. ユーザー応答
   ├→ キャンセル → 処理中止
   └→ OK → 続行
4. DELETE /api/todos/{id}
5. 成功時
   ├→ 現在のフィルタ状態を維持してリスト再取得
   └→ loadStats() で統計更新
```

---
