# EV-002-04: フィルタ切替

> **ドキュメント種別**: 最新仕様（継続的にメンテナンス）

## 1. 基本情報

| 項目 | 内容 |
|------|------|
| イベントID | EV-002-04 |
| イベント名 | フィルタ切替 |
| 画面 | SC-002: ToDoリスト画面 |
| トリガー | フィルタボタン（すべて/未完了/完了済み）クリック |

---

## 2. 処理フロー

```
1. フィルタボタンクリック
2. ボタンのアクティブ状態を更新
3. フィルタ種別に応じてAPI呼び出し
   ├→ すべて: GET /api/todos
   ├→ 未完了: GET /api/todos?completed=false
   └→ 完了済み: GET /api/todos?completed=true
4. レスポンスでカードを再描画
```

---

## 3. フィルタ種別

| ボタン | APIパラメータ |
|--------|--------------|
| すべて | なし |
| 未完了 | ?completed=false |
| 完了済み | ?completed=true |

---

## 4. 使用API

| API | メソッド | リクエスト | レスポンス |
|-----|---------|-----------|-----------|
| /api/todos | GET | クエリパラメータ(optional) | ToDo配列 |
