# EV-002-01: 初期読込

> **ドキュメント種別**: 最新仕様（継続的にメンテナンス）

## 1. 基本情報

| 項目 | 内容 |
|------|------|
| イベントID | EV-002-01 |
| イベント名 | 初期読込 |
| 画面 | SC-002: ToDoリスト画面 |
| トリガー | DOMContentLoaded（ページロード完了時） |

---

## 2. 処理フロー

```
1. ページロード完了
2. loadTodos() 呼び出し
   └→ GET /api/todos
   └→ レスポンスを受け取りカード描画
3. loadStats() 呼び出し
   └→ GET /api/todos/stats
   └→ 統計エリア更新
```

---

## 3. 使用API

| API | メソッド | リクエスト | レスポンス |
|-----|---------|-----------|-----------|
| /api/todos | GET | なし | ToDo配列 |
| /api/todos/stats | GET | なし | { total, completed, pending } |

---

## 4. エラー処理

| エラー | 対応 |
|--------|------|
| 通信失敗 | エラーメッセージを表示（5秒後消去） |
