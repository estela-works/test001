## 5. 使用API一覧

| 画面 | 操作 | API | メソッド |
|------|------|-----|---------|
| ToDoリスト | 一覧取得 | /api/todos | GET |
| ToDoリスト | フィルタ取得 | /api/todos?completed={bool} | GET |
| ToDoリスト | 統計取得 | /api/todos/stats | GET |
| ToDoリスト | 新規作成 | /api/todos | POST |
| ToDoリスト | 完了切替 | /api/todos/{id}/toggle | PATCH |
| ToDoリスト | 削除 | /api/todos/{id} | DELETE |
