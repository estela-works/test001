# EV-003-04: チケット一覧遷移

> **ドキュメント種別**: 最新仕様（継続的にメンテナンス）

## 1. 基本情報

| 項目 | 内容 |
|------|------|
| イベントID | EV-003-04 |
| イベント名 | チケット一覧遷移 |
| 画面 | SC-003: プロジェクト画面 |
| トリガー | チケット一覧ボタンクリック |

---

## 2. 処理フロー

```
1. ボタンクリック
2. 遷移先URL構築
   └→ 案件カード: /todos.html?projectId={projectId}
   └→ 未分類カード: /todos.html?projectId=none
3. window.location.href で遷移
```

---

## 3. 遷移先

| 遷移元 | 遷移先URL | 説明 |
|--------|----------|------|
| 案件カード | /todos.html?projectId={id} | 特定案件のチケット一覧 |
| 未分類カード | /todos.html?projectId=none | 案件未設定のチケット一覧 |
