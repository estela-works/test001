# EV-003-02: 案件作成

> **ドキュメント種別**: 最新仕様（継続的にメンテナンス）

## 1. 基本情報

| 項目 | 内容 |
|------|------|
| イベントID | EV-003-02 |
| イベント名 | 案件作成 |
| 画面 | SC-003: プロジェクト画面 |
| トリガー | 作成ボタンクリック / Enterキー（案件名入力欄） |

---

## 2. 処理フロー

```
1. 入力値取得
   └→ 案件名（必須）
   └→ 説明（任意）
2. バリデーション
   └→ 案件名が空の場合 → エラー表示
3. API呼び出し
   └→ POST /api/projects
4. 成功時
   └→ フォームクリア
   └→ loadProjects() で一覧再取得
5. 失敗時
   └→ エラーメッセージ表示
```

---

## 3. 使用API

| API | メソッド | リクエスト | レスポンス |
|-----|---------|-----------|-----------|
| /api/projects | POST | { name, description } | 作成されたProject |

---

## 4. バリデーション

| 項目 | ルール | エラーメッセージ |
|------|--------|----------------|
| 案件名 | 必須（空白不可） | 「案件名を入力してください。」 |

---

## 5. エラー処理

| エラー | 対応 |
|--------|------|
| バリデーションエラー | エラーメッセージ表示（5秒後消去） |
| API失敗 | 「案件の作成に失敗しました。」表示（5秒後消去） |
| ネットワークエラー | 「ネットワークエラーが発生しました。」表示（5秒後消去） |
