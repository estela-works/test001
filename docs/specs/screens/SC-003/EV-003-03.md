# EV-003-03: 案件削除

> **ドキュメント種別**: 最新仕様（継続的にメンテナンス）

## 1. 基本情報

| 項目 | 内容 |
|------|------|
| イベントID | EV-003-03 |
| イベント名 | 案件削除 |
| 画面 | SC-003: プロジェクト画面 |
| トリガー | 削除ボタンクリック |

---

## 2. 処理フロー

```
1. 確認ダイアログ表示
   └→ 「この案件を削除しますか？配下のチケットもすべて削除されます。」
2. キャンセル時
   └→ 処理中断
3. OK時
   └→ DELETE /api/projects/{id}
4. 成功時
   └→ loadProjects() で一覧再取得
5. 失敗時
   └→ エラーメッセージ表示
```

---

## 3. 使用API

| API | メソッド | リクエスト | レスポンス |
|-----|---------|-----------|-----------|
| /api/projects/{id} | DELETE | なし | 204 No Content |

---

## 4. 確認ダイアログ

| 項目 | 内容 |
|------|------|
| メッセージ | 「この案件を削除しますか？配下のチケットもすべて削除されます。」 |
| OK | 削除実行 |
| キャンセル | 処理中断 |

---

## 5. エラー処理

| エラー | 対応 |
|--------|------|
| API失敗 | 「案件の削除に失敗しました。」表示（5秒後消去） |
| ネットワークエラー | 「ネットワークエラーが発生しました。」表示（5秒後消去） |
