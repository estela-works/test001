# 画面詳細仕様

各画面の詳細仕様を格納するフォルダ。

## フォルダ構成

```
screens/
├── index.md                 # 画面インデックス
├── SC-001/                  # ホーム画面
│   ├── screen.md            # 画面仕様
│   ├── events.md            # イベント一覧
│   └── EV-001-XX.md         # イベント詳細
└── SC-002/                  # ToDoリスト画面
    ├── screen.md            # 画面仕様
    ├── events.md            # イベント一覧
    └── EV-002-XX.md         # イベント詳細
```

## 画面一覧

| ID | 画面名 | 説明 |
|----|--------|------|
| [SC-001](SC-001/) | ホーム画面 | トップページ |
| [SC-002](SC-002/) | ToDoリスト画面 | ToDo管理画面 |

## ドキュメント構造

| 階層 | ファイル | 内容 |
|------|---------|------|
| 1 | screen.md | 画面基本情報、レイアウト、構成要素 |
| 2 | events.md | イベント一覧（サマリー） |
| 3 | EV-XXX-XX.md | イベント詳細（処理フロー、API、エラー処理） |
