# EV-005-08: チケット削除

> **ドキュメント種別**: 最新仕様（継続的にメンテナンス）

## 1. 基本情報

| 項目 | 内容 |
|------|------|
| イベントID | EV-005-08 |
| イベント名 | チケット削除 |
| 画面 | SC-005: チケット一覧画面 |
| トリガー | TodoTableからのdeleteイベント |

---

## 2. 処理フロー

```
1. TodoTableからチケットID受信
2. handleDelete(id) 実行
3. confirm('このチケットを削除しますか？') 表示
   ├── キャンセル → 処理終了
   └── OK → 続行
4. todoStore.deleteTodo(id) 呼び出し
5. DELETE /api/todos/{id} 実行
6. ストア更新
7. 表示更新
```

---

## 3. 使用API

| API | メソッド | リクエスト | レスポンス |
|-----|---------|-----------|-----------|
| /api/todos/{id} | DELETE | なし | 204 No Content |
