## 2. 処理フロー

```
1. TodoTableからソートキー受信
2. handleSortChange(key) 実行
3. 同じカラムの場合
   └→ sortOrder.value = sortOrder.value === 'asc' ? 'desc' : 'asc'
4. 別カラムの場合
   ├→ sortKey.value = key
   └→ sortOrder.value = 'asc'
5. computed（filteredAndSortedTodos）が再計算
6. 表示更新
```
