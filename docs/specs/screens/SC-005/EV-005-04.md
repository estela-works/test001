---

## 2. 処理フロー

```
1. TodoTableFilterから新しいフィルタ条件受信
2. handleFilterChange(newFilter) 実行
3. filter.value = { ...newFilter }
4. computed（filteredAndSortedTodos）が再計算
   ├── 完了状態フィルタ適用
   ├── 担当者フィルタ適用
   ├── 案件フィルタ適用
   ├── 開始日期間フィルタ適用
   └── 終了日期間フィルタ適用
5. 表示更新
```

---

## 3. フィルタ項目

| 項目 | キー | 値 | 初期値 |
|------|-----|-----|--------|
| 完了状態 | completed | 'all' / 'pending' / 'completed' | 'all' |
| 担当者 | assigneeId | null / number | null |
| 案件 | projectId | null / number | null |
| 開始日From | startDateFrom | null / string | null |
| 開始日To | startDateTo | null / string | null |
| 終了日From | dueDateFrom | null / string | null |
| 終了日To | dueDateTo | null / string | null |

---

## 4. フィルタ適用順序

1. キーワード検索
2. 完了状態
3. 担当者
4. 案件
5. 開始日From
6. 開始日To
7. 終了日From
8. 終了日To
