# EV-004-01: 初期読込

> **ドキュメント種別**: 最新仕様（継続的にメンテナンス）

## 1. 基本情報

| 項目 | 内容 |
|------|------|
| イベントID | EV-004-01 |
| イベント名 | 初期読込 |
| 画面 | SC-004: ユーザー管理画面 |
| トリガー | window.onload（ページロード完了時） |

---

## 2. 処理フロー

```
1. ページロード完了
2. loadUsers() 呼び出し
   └→ GET /api/users
   └→ レスポンスを受け取りカード描画
3. updateStats() 呼び出し
   └→ users.length を統計エリアに表示
```

---

## 3. 使用API

| API | メソッド | リクエスト | レスポンス |
|-----|---------|-----------|-----------|
| /api/users | GET | なし | User配列 |

---

## 4. エラー処理

| エラー | 対応 |
|--------|------|
| 通信失敗 | 「ユーザーリストの読み込みに失敗しました。」表示（5秒後消去） |
| ネットワークエラー | 「ネットワークエラーが発生しました。」表示（5秒後消去） |
