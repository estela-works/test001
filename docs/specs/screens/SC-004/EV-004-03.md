# EV-004-03: ユーザー削除

> **ドキュメント種別**: 最新仕様（継続的にメンテナンス）

## 1. 基本情報

| 項目 | 内容 |
|------|------|
| イベントID | EV-004-03 |
| イベント名 | ユーザー削除 |
| 画面 | SC-004: ユーザー管理画面 |
| トリガー | 削除ボタンクリック |

---

## 2. 処理フロー

```
1. 確認ダイアログ表示
   └→ 「このユーザーを削除しますか？担当中のチケットは「未割当」になります。」
2. キャンセル時
   └→ 処理中断
3. OK時
   └→ DELETE /api/users/{id}
4. 成功時
   └→ loadUsers() で一覧再取得
5. 失敗時
   └→ エラーメッセージ表示
```

---

## 3. 使用API

| API | メソッド | リクエスト | レスポンス |
|-----|---------|-----------|-----------|
| /api/users/{id} | DELETE | なし | 204 No Content |

---

## 4. 確認ダイアログ

| 項目 | 内容 |
|------|------|
| メッセージ | 「このユーザーを削除しますか？担当中のチケットは「未割当」になります。」 |
| OK | 削除実行 |
| キャンセル | 処理中断 |

---

## 5. エラー処理

| エラー | 対応 |
|--------|------|
| API失敗 | 「ユーザーの削除に失敗しました。」表示（5秒後消去） |
| ネットワークエラー | 「ネットワークエラーが発生しました。」表示（5秒後消去） |
