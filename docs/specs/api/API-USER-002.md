## 5. レスポンス

### 5.1 ステータスコード

| コード | 条件 |
|--------|------|
| 200 | 成功 |
| 404 | 指定IDが存在しない |

### 5.2 レスポンスボディ（成功時）

```json
{
  "id": 1,
  "name": "山田太郎",
  "createdAt": "2025-12-22T10:00:00"
}
```

---

## 6. 処理フロー

```
1. リクエスト受信（パスパラメータ id 取得）
   │
2. UserMapper.findById(id) 呼び出し
   │
   ├─ 存在する → 3. レスポンス返却（200 + User）
   │
   └─ 存在しない → 3. レスポンス返却（404）
```
