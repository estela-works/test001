## 5. レスポンス

### 5.1 ステータスコード

| コード | 条件 |
|--------|------|
| 200 | 成功 |

### 5.2 レスポンスボディ

```json
[
  {
    "id": 1,
    "title": "タスク名",
    "description": "説明",
    "completed": false,
    "createdAt": "2025-12-22T10:00:00",
    "assigneeId": 1,
    "assigneeName": "山田太郎"
  }
]
```

---

## 6. 処理フロー

```
1. リクエスト受信（completed パラメータ取得）
   │
2. TodoMapper.findByCompleted(completed) 呼び出し
   │
3. 条件に合致するレコードを createdAt DESC でソート
   │
4. レスポンス返却（200 + List<Todo>）
```
