## 5. レスポンス

| コード | 条件 |
|--------|------|
| 200 | 成功 |

### 5.2 レスポンスボディ

```json
[
  {
    "id": 1,
    "name": "山田太郎",
    "createdAt": "2025-12-22T10:00:00"
  }
]
```

| フィールド | 型 | 説明 |
|-----------|-----|------|
| id | Long | 一意識別子 |
| name | String | ユーザー名 |
| createdAt | LocalDateTime | 作成日時 |

---

## 6. 処理フロー

```
1. リクエスト受信
   │
2. UserMapper.findAll() 呼び出し
   │
3. 全件を createdAt ASC でソートして取得
   │
4. レスポンス返却（200 + List<User>）
```
