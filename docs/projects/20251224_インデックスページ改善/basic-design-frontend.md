## 1. 概要

### 1.1 本設計書の目的

ホーム画面（SC-001）のフロントエンド基本設計を定義する。既存画面（SC-002〜SC-004）と統一されたカードベースUIへのリニューアルを行う。

### 1.2 フロントエンドの役割

| 責務 | 説明 |
|------|------|
| 画面表示 | アプリケーションタイトルとナビゲーションカードの表示 |
| ユーザー操作 | カードクリックによる画面遷移 |
| API連携 | なし（静的ページ） |

## 5. 共通UI要件

### 5.1 共通コンポーネント

| コンポーネント | 用途 |
|--------------|------|
| ナビゲーションカード | 各機能への遷移リンク（カード形式） |
| コンテナ | 中央寄せレイアウト（max-width: 800px） |

### 5.2 UI状態

| 状態 | 表示 |
|------|------|
| ローディング中 | 該当なし（静的ページ） |
| エラー発生時 | 該当なし（静的ページ） |
| データなし | 該当なし（静的ページ） |
| ホバー時 | カードが右に移動、左にテーマカラーボーダー表示 |
