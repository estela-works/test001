## 2. 変更なしの理由

本案件はフロントエンドのみの改善（静的HTMLの追加）であり、以下の理由によりSQL変更は不要である:

1. **データアクセス不要**: ホーム画面はデータベースへのアクセスを行わない
2. **静的ナビゲーション**: 動的データの表示がない
3. **Mapper変更なし**: 既存のMapperインターフェース・XMLに変更なし
