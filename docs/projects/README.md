# 案件スコープドキュメント

案件ごとの設計ドキュメントを格納するフォルダ。

## フォルダ構成

```
projects/
├── YYYYMM_案件名/           # 案件ごとのフォルダ
│   ├── requirements.md      # 要件整理書
│   ├── basic-design-*.md    # 基本設計書
│   ├── detail-design-*.md   # 詳細設計書
│   └── test-spec*.md        # テスト仕様書
└── template/                # テンプレート
```
