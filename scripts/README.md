# ユーティリティスクリプト

プロジェクト管理用のユーティリティスクリプト集。

## スクリプト一覧

| スクリプト | 説明 |
|-----------|------|
| `check-missing-readme.ps1` | READMEが不足しているフォルダを検出（Windows PowerShell） |
| `check-missing-readme.sh` | READMEが不足しているフォルダを検出（Unix/Linux） |
| `check-project-status.sh` | プロジェクトの状態をチェック |

## 使用方法

### Windows (PowerShell)

```powershell
.\scripts\check-missing-readme.ps1
```

### Unix/Linux/Mac

```bash
chmod +x scripts/*.sh
./scripts/check-missing-readme.sh
```
